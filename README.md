# React Base - Curso de React (2019 -> 2026)

Este projeto faz parte dos meus estudos de **React.js**. O objetivo é seguir um cronograma de aprendizado robusto (original de 2019), mas adaptando as ferramentas para as versões mais atuais do mercado (**React 19**).

## 🚀 Tecnologias Utilizadas

- **React 19** (Versão estável de 2026)
- **Styled Components** (CSS-in-JS para estilização)
- **React Router Dom v5** (Para navegação entre páginas)
- **Git/GitHub** (Versionamento e Deploy)

## 🛡️ Foco em DevSecOps

Durante o desenvolvimento, estou aplicando boas práticas de segurança e infraestrutura, como:

- Gestão limpa de dependências (evitando pacotes obsoletos).
- Configuração de ambiente em ambiente **Linux (Ubuntu)**.
- Organização de commits seguindo padrões profissionais.
